# CT&DT Galgotias - Batch No: 5
## Task#01 – Product Failure Analysis (Machine Failure Analysis)

## 📋 Project Objective
This repository documents our diagnostic data analysis for Task 1 under the Fundamentals of Design Thinking curriculum framework. Utilizing predictive engineering datasets, this study explores operational variables, stress thresholds, and systematic design vulnerabilities that lead to physical mechanical failure states in industrial equipment.

## ⚙️ Explored Feature Vectors
Based on the foundational Kaggle engineering dataset parameters, our analysis isolates:
* **Thermal Differentials:** Monitoring how ambient vs. internal process temperatures impact unit degradation.
* **Mechanical Torque Stress:** Tracking the rotational speed variations (RPM) alongside torque limits (Nm).
* **Tool Wear Lifecycles:** Evaluating component aging parameters that act as direct precursors to systemic failures.
